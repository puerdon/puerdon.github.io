使用data宣告全新的type

與type declaration只是宣告一個「關於既存類型的同義詞」不同，使用data關鍵字可以宣告全新的類型。
在Prelude中，Bool類型即是以下列方式宣告的：

type String = [Char]

type Point = (Int, Int)

type Transformation = Point -> Point

type Pair a = (a, a)
`
一個Pair可以是Pair Int (Int, Int)，可能是(Char, Char)
