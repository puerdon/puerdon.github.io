Data declaration: 宣告全新的type

與type declaration只是宣告一個「關於既存類型的同義詞」不同，使用data關鍵字可以宣告全新的類型。
在Prelude中，Bool類型即是以下列方式宣告的： 

data Bool = True | False
  • 在=左邊的稱為 type constructor (上例中的Bool)
  • 在=右邊的稱為 data constructor 或 value constructor (上例中的True 或 False)
